Current Challenges

Custom Button Widgets

⏱️ 20-30 minutes

Create three different custom button styles that showcase Flutter styling techniques. Learn to build filled, outlined, and custom styled buttons.

You'll learn: ElevatedButton, OutlinedButton, ButtonStyle, Colors, BorderRadius, Widget styling
2

Profile Card Widget

⏱️ 25-35 minutes

Build a personal profile card with avatar, name, bio, and social links. Master layout fundamentals and create professional-looking UI components.

You'll learn: Container, Card, Row, Column, Text styling, Icons, Padding, Layout hierarchy
3

Simple Todo List

⏱️ 35-45 minutes

Create a functional todo app with add, complete, and delete functionality. Dive into state management and dynamic lists - core Flutter concepts!

You'll learn: StatefulWidget, setState(), ListView.builder, TextEditingController, FloatingActionButton, Dynamic lists
